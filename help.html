Getting started
General

Thank you for installing iBees. iBees consists of a hub, with up to 5 wireless scales connected to it. Once setup it will run fully automatical. At the end of everyday it will upload its data to the internet. Various alerts can be armed with a simple SMS.

SIM card

The system uploads its data through a GPRS/G2 datalink. It requires a SIM card with a SMS and data subscription. The system uses a univeral PIN (9876). Before inserting the SIM card into the hub, change its PIN to 9876 and test network connection. A daily upload will only use about 20kB of data. This is less than 1MB per month!

Hub

Open the plastic casing, insert the SIM-card and switch to ON. The hub is best placed in the shadow, as it contains a temperature sensor. To establish an excellent wireless scale connection, avoid having dense obstacles between the scales and hub.

Scale

Open the plastic casing, make sure the battery is in place and switch to ON.

Synchronization

Scales synchronize automatically with the hub. They periodically wake-up, transmit their data and go back to sleep. The hub aims to upload its stored data between 8pm and 9pm. If the first upload attempt was unsucessfull, it will try again after the programmed interval period. When starting up a new hub or replacing its battery's, the system is first disoriented. It has to recalibrate its internal clock. This normally takes up to 3 days.

SMS control
General

The system can be controlled through sending simple SMS messages. When the hub uploads data, it first checks for received messages. You can send an SMS to the system any time of the day. The SMS will however only be read at the end of the day. You can arm/disarm a swarm-alert, a theft-alert or a trigger weight. You can change the wake-up interval or contact phone number. You can even get daily SMS reports. Caution: the control SMS must be spelled exactly as below! Commands are case-sensitive. A command will only be followed if a confirmation SMS was received.

Swarm-Alert

When armed, the system will send a swarm-alert SMS in case of a sudden weight drop on one of the scales. A swarm-alert can be armed or disarmed at any time. It will however only send out an SMS when the weight drop happens between 10am adn 6pm during the months May, June, July and August. Valid commands are: SwarmAlert ON, SwarmAlert OFF

Theft-Alert

When armed, the system will send a theft-alert in case of sudden big weight drop between 7pm and 9am. Valid commands are: TheftAlert ON, TheftAlert OFF

Interval

The systems wake-up periodically to measure the scale weight, and than goes back to sleep. The wake-up interval can be set at 15min, 30min or 60min. A 30min interval creates 48 datapoints per day. Interval period could for instance be increase during the winter. Valid commands are: Interval 15min, Interval 30min, Interval 60min

Trigger

Any iBees scale can be individually programmed to send you a SMS when its weight drops below a certain value. This can be handy during the winter months when the hive requires more feeding. To set scale 1 value to 30kg and scale 2 to 25kg send: Trigger 1=30kg, 2=25kg To disarm just set the trigger value to 0.

SMS-report

iBees can send you a daily SMS with the weight changes of the various scales. This makes it possible to follow up the changes without internet access. Unfortenately due to limited memory, daily reports are only possible with 30min or 60min intervals. Valid commands are: SMSreport ON, SMSreport OFF

New mobile number

iBees can be reprogrammed to accept a new mobile number. This new number will than be used to control and communicate with the system. Warning: be 100% sure of the new number! Some changes can not be undone. Valid command are: NewNbr +32478642330

Battery
Low voltage

Both hub and scales are using 18650 rechargeable lithium battery's. When the scale or hub voltage drops to low, the system will send out a 'Low voltage" warning. When the warning is ignored and the voltage drops below 3,55V the scale or hub stop working. Caution: Do not fully discharge a lithium battery as it will be damaged!

Charging

Set the switch to OFF and remove the battery from its clips. A fully charged battery has an output of about 4,1V. A lithium battery can be over 500 times recharged.

FAQ
Communication cost

Monthly data upload for a standard 30min reading interval will be less than 1MB. Apart form occasional alert, there will be normally no SMS communication (unless daily SMS reports are requested). Actual communication cost will of course vary per provider.

Wireless scale range

Obstructions between scale and hub will reduce the communication range. The hub should be placed in the upright possition for best antenna orientation. Unobstructed scales will have a range of 50m.

Working on hives

Working on a hive might trigger a swarm-alert or theft-alert. This can be avoided by disarming both alerts by sending an SMS on the previous day. Switching the scale or hub to OFF will not prevent the alert, but will merely delay it until switch back ON.
